Help

On Start Page.

You must do the following to chat to another user.

These two first in any order.

  1. Choose whether you are a Supporter or you need Support
  2. Select what you want to talk about

Then click to Find a Match and Start Chatting.

On Chat Page.

When matched with a user you can do the following in any order.

  • You can Type into and Send a text Message by pressing Enter or .
  • You can Skip the matched user by clicking on .
  • You can Report the matched user by clicking on .
  • You can invite the other user to a game, to choose a game click then click on your chosen game by clicking on choosing game button like .

If you skip, report, get skipped, reported or blocked you will be redirected to start page where you can change your preferences and match with another user to start chatting again.
